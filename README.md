
# Maize Disease Expert System

Hi! This is a **Maize Disease Expert System** made with Python and Streamlit. It helps farmers or students identify maize diseases by selecting symptoms and conditions.

## Features
- Interactive app where you select symptoms and environmental conditions.  
- Diagnoses many maize diseases like MLND, Rust, Blight, Smut, and more.  
- Shows possible risk factors and disease co-occurrences.  
- Forward chaining logic recommends diseases instantly.

## How to Run
1. Clone the repository:

git clone https://github.com/Ally622/Maize_diseases_system.git
cd Maize_diseases_system
2. Install dependencies:

pip install -r requirements.txt
3. Run the app:
streamlit run maize_diseases.py
Pick symptoms and conditions from the lists.

The app will show possible diseases that match your selections.
